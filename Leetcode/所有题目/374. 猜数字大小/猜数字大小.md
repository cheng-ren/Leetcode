#  [374. 猜数字大小](https://leetcode.cn/problems/guess-number-higher-or-lower/)

猜数字游戏。你需要猜出由系统选出的数字 `pick`，数字范围在 `1` 到 `n` 之间（包含两端）。

每次猜测后，会通过接口 `guess(num)` 返回以下结果：

- 如果 `num < pick`，返回 `1`
- 如果 `num > pick`，返回 `-1`
- 如果 `num == pick`，返回 `0`

请你返回正确猜到的数字。


**示例 1:**

```
输入: n = 10, pick = 6
输出: 6
```

**示例 2:**

```
输入: n = 1, pick = 1
输出: 1
```

**示例 3:**

```
输入: n = 2, pick = 1
输出: 1
```


**提示:**

- `1 <= n <= 2^31 - 1`
- `1 <= pick <= n`
