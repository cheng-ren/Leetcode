# [257. 二叉树的所有路径](https://leetcode.cn/problems/binary-tree-paths/)

给你一个二叉树的根节点 `root` ，按任意顺序，返回所有从根节点到叶子节点的路径。

叶子节点是指没有子节点的节点。

**示例 1：**

给定二叉树：

```
   1
 /   \
2     3
 \
  5
```

输出：

```
["1->2->5", "1->3"]
```

**示例 2：**

```
输入：root = [1,2,3,null,5]
输出：["1->2->5","1->3"]
```

**提示：**

- 树中节点数目在范围 `[0, 100]` 内
- `-100 <= Node.val <= 100`
