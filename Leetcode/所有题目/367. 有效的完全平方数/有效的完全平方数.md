#  [367. 有效的完全平方数](https://leetcode.cn/problems/valid-perfect-square/)

给你一个正整数 `num` 。如果 `num` 是一个完全平方数，则返回 `true` ，否则返回 `false` 。

**进阶：**不要使用任何内置的库函数，如 `sqrt`。

**示例 1：**

```
输入：num = 16
输出：true
```

**示例 2：**

```
输入：num = 14
输出：false
```

**示例 3：**

```
输入：num = 1
输出：true
```

**提示：**

- `1 <= num <= 2^31 - 1`
